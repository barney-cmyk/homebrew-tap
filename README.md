# Neuron Flame Homebrew Tap

Official [Homebrew](https://brew.sh) tap for [Neuron Flame](https://neuronflame.com) — voice-to-text for macOS, 100% local, free, GPL-3.

## Install

```sh
brew install barney-cmyk/tap/neuron-flame
```

## Upgrade

```sh
brew upgrade --cask neuron-flame
```

(The app also self-updates via Sparkle — both paths deliver the same signed, Apple-notarised build.)
